Héroes del código es un proyecto de ejemplo la cuál consta de ser una academia en línea para programadores, además de ofrecer el apoyo de una comunidad de desarrolladores de apoyo para las personas
que están comenzando en el mundo de la programación.

No cuento con el derecho de las imágenes e íconos usados en este proyecto, ya que solo fue realizado con
fines de aprendizaje.

Esta es la primera versión de la página, próximamente se añadirán más elementos como animaciones y 
optimizaciones conforme vaya adquiriendo aprendizaje.

Este proyecto tiene una base responsive mobile a base de media queries.
